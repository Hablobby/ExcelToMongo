# ExcelToMongo


# To run the project

1. Clone the project
2. Create a virtual environment (python -m venv venv)
3. Activate the virtual environment (source ./venv/Scripts/activate)
4. Install the requirements (pip install -r requirements.txt)

Running the Backend
